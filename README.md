# Realm

**Private chat application — your server, your data.**

---

## Download

Go to the [Releases](https://github.com/nefesec/realm/releases) page and download the version for your platform:

| Platform | File |
|---|---|
| Windows | `Realm-Setup.exe` |
| Linux | `Realm.AppImage` |
| Android | `Realm.apk` |

---

## Installation

### Windows
Run `Realm-Setup.exe` and follow the installer.

### Linux
```bash
chmod +x Realm.AppImage
./Realm.AppImage
```

### Android
Enable installation from unknown sources in your settings, then open `Realm.apk`.

---

## First launch

On first launch, enter the address of your Realm server to connect.
